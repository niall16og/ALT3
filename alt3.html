Key Aims and Objectives

In our project our main aim is to predict the probability of the outcome of a game between 2 teams (Man United & Liverpool). We want to be able to retrieve data on their past 5 results and compare them to determine what the outcome would be between the 2 teams.

Objectives:

  • Create code that enables us to retrieve results.
  • Calculate the form based on the results
  • Compare the results
  • Tell the user who the better team is based on the previous results

Our Project


Modelling and Simulation Idea

The modelling and simulation idea that we had was a model to simulate the match result between two teams to predict the most likely outcome, (win, lose or draw), based on various factors of previous performance, such as recent form.

What we chose

The model we have made is a statistical model that operates on various factors in the past performance of both teams, analysing them to produce an output.

We have chosen this model idea because we find it to be an interesting area to explore and believe it will present some interesting challenges that will both be difficult but achievable to overcome.

The primary thing we are focusing on in this model is the recent form of each team over the five previous games and applying certain weightings to each game.

The primary scenario would be when one team has a better form than the other team, then the model would show them as the most likely to win. Another example of a scenario would be if both teams have an equal form, then the model should show the most likely outcome as a draw.

The system will fetch the recent games of each team and assign different weightings to their result. Based on this each team will receive a form rating and the probability of each outcome of the match is calculated based on these ratings.

Pseudocode

Fetch data
For each match - apply weighting to result based on recency
Calculate form ratings
Incorporate points per game and previous head-to-head with form ratings
Compare form ratings and determine most likely result
Calculate probabilities for each outcome
Output results

Reflection

Looking back on our project there are obviously things we would have done differently. If we were to do it again we would have probably altered the code so that it could be used on many different teams instead of 2 set ones. Also, if we had more time on our Alt we would have added more different and complex variables to our code. Another element we could have changed is having more user interaction. Using the code the user is only asked to input the team names and they have the results, we could have also got them to save their results and compare them later on in the year if we had more time.
